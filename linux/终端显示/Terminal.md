|        快捷键        |    操作    |

| ctrl+shift+c         | 复制       |
| ctrl+shift+v         | 粘贴       |

### [当前用户@当前主机名 当前所在位置]#root/$user
|          命令         |        作用        |
|-----------------------|--------------------|
| clear                 | 清空终端           |
| history               | 显示终端命令记录   |
| startx &              | 启动图形界面       |
| Ctrl+Alt+f1           | 虚拟终端           |
|                       | f2~f6，多个终端    |
| Ctrl+Alt+f7           | 图形界面           |
| 图形界面open Terminel | 打开模拟终端       |
| shut down             | 关机（同windows）  |
| reboot                | 重启               |
