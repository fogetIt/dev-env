用户信息
-------
    :id:     显示当前用户信息
    :id xxx: 显示指定用户信息

        -g       显示用户所属主组 ID
        --group  显示用户所属主组 ID
        -G       显示用户所属组 ID
        --groups 显示用户所属组 ID
        -n       显示用户所属组名称
        --name   显示用户所属组名称
        -r       显示实际 ID
        --real   显示实际 ID
        -u       显示用户 ID
        --user   显示用户 ID
        -help    显示帮助
        -version 显示版本信息
    :groups:     显示当前用户所属组名称
    :groups xxx: 显示指定用户所属组名称
    :who:        显示用户名、终端、登录时间
    :who am i:   显示用户名、终端、登录时间（简化）
    :whoami:     显示当前登录用户名
    :users:      显示当前登录用户名


w(ho is doing what?)
--------------------
    - 显示当前登录用户的更多信息


s(witch )u(ser)
---------------
    - 切换用户
        :``#``: root
        :``$``: 普通用户
        -l  完全切换


uid
----
    :root: 0
    :系统用户: 1~999

        - 系统用户，使用服务或进程时的用户
    :普通用户: >=1000
