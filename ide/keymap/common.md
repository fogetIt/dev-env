### 阅读
|快捷键       |作用          |特殊 |
|:-----------|:------------|:---|
|CTRL -      |折叠当前代码   |    |
|CTRL +      |展开当前代码   |    |
|CTRL SHIFT -|折叠所有代码   |    |
|CTRL SHIFT +|展开所有代码   |    |
|            |             |    |
|CTRL /      |注释、取消注释行|    |
|Ctrl+Shift+/|注释多行      |idea|

#### 搜索
|快捷键     |作用       |
|:---------|:---------|
|Ctrl+F    |关键字查找 |
|Ctrl+H    |查找替换   |
|Ctrl+alt+F|文件夹内查找|
|Ctrl+P    |文件名查找 |


#### 缩进
|快捷键    |作用    |特殊    |
|:--------|:------|:------|
|Tab      |向右缩进|       |
|Shift+Tab|向左缩进|sublime|


### 书签
|快捷键                |作用                         |特殊    |
|:--------------------|:---------------------------|:------|
|CTRL+F11             |设置书签                     |       |
|F2/shift+F2          |切换书签                     |sublime|
|SHIFT+F11            |显示所有书签                  |       |
|                     |                            |       |
|CTRL F12//Ctrl+P -> @|列出文件方法LIST              |       |
|                     |除了用上下箭选择外，还可以输入字母|       |

### 选择
|快捷键         |作用                      |特殊|
|:-------------|:------------------------|:--|
|Ctrl+←/→      |逐词移动                  |   |
|Ctrl+Shift+←/→|逐词选择                  |   |
|              |                         |   |
|              |                         |   |
|Ctrl+D        |选中光标所占的文本          |   |
|              |继续操作则会选中下一个相同文本|   |
|              |Ctrl+K会跳过一个          |   |
|Ctrl+Shift+D  |未选中时，复制当前行到下一行  |   |
|              |选中时复制粘贴选中部分      |   |
|Ctrl+A        |选中全部                  |   |

### 显示
|快捷键               |作用             |特殊    |
|:-------------------|:---------------|:------|
|Ctrl+Tab            |切换当前窗口      |       |
|Ctrl+W              |关闭当前窗口      |       |
|Ctrl+PageUp/PageDown|向左/右切换当前窗口|sublime|

### 编辑多行
|快捷键        |作用              |特殊    |
|:------------|:----------------|:------|
|ctrl+左键     |点击多处同时添加光标|sublime|
|Alt+Shift+左键|点击多处同时添加光标|idea   |
|             |                 |       |
|Ctrl+X       |剪切整行          |       |
|Ctrl+C       |复制整行          |       |
|Ctrl+V       |粘贴整行          |       |
|Ctrl+Z       |撤销              |       |
|Ctrl+Y       |恢复撤销          |       |
